Spark · Examples

Every example shipped with Spark, with one-click links to run it through the Three.js, A-Frame, or BabylonJS backend.

● engine-aware: the example honours ?engine=three|aframe|babylon and renders through the selected backend. ○ exception/showcase: either a headset-only XR/editor surface that is intentionally outside the backend smoke gate, or a Babylon host reference page that is already Babylon-specific. See src/backends/README.md and MULTI-BACKEND-PARITY-PLAN.md for the rollout plan.

A-Frame parity uses a structural AFRAME mock (npm/CDN A-Frame bundles its own Three.js fork and renders blank against Spark's). Engine-aware example pages use Babylon's texture bridge by default; the spark-babylon-native reference page demonstrates native Babylon mesh depth-sorting.
